# Health Management System

A Python OOP-based CLI application for managing hospital operations including patients, doctors, and appointments.

## Features
- Add/View/Update/Delete Patients and Doctors
- Schedule/View/Cancel Appointments
- Data persistence with JSON

## Usage
```bash
python main.py
```

## Files
- `main.py`: Entry point and menu
- `hospital.py`: Main manager class
- `patient.py`, `doctor.py`, `appointment.py`: Model classes
- `utils.py`: Data handling

Built for educational purposes.
